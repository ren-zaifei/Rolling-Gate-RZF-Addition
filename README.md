# 卷帘门附属| RGRA
这是一个卷帘门附属，规则宗旨是让游戏更简单。
## 前置
必需：卷帘门模组 [地址](https://www.mcmod.cn/class/18118.html)
## 规则
| 规则                    | 类型        | 默认值     | 允许值             | 描述                 |
|-----------------------|-----------|---------|-----------------|--------------------|
| `cutString`           | `boolean` | `false` | `false`, `true` | `引入fabric的刷线机`     |
| `lavaAndesiteHandler` | `boolean` | `false` | `false`, `true` | `使岩浆碰到铁块后凝固并生成安山岩` |
